Release Checklist

Content

标题、描述、卡片文案、链接分类一致，不放无关入口。

Mobile

手机宽度不横向溢出，按钮可点击，代码块自动换行。

Network

图片、CSS、JS 都走相对路径；404 和缓存问题部署后检查。

DNS

解析到目标平台，权威 DNS 与公共 DNS 都能查到记录。

HTTPS

证书签发后再开启强制 HTTPS；没证书时保留 HTTP 可用。

Archive

保留源码、部署脚本、域名记录和变更说明，方便回滚。
